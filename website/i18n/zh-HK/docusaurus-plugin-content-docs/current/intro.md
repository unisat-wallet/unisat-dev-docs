# 教學介紹

讓我們在 **5 分鐘內認識 Docusaurus**。

## 開始使用

透過 **建立一個全新的網站** 來開始吧。

或者直接透過 **[docusaurus.new](https://docusaurus.new)** **立即體驗 Docusaurus**。

### 你需要準備的環境

- [Node.js](https://nodejs.org/en/download/) 版本 **20.0 或以上**：
  - 在安裝 Node.js 時，建議勾選所有與相依套件（dependencies）相關的選項。

## 產生一個新網站

使用 **classic 範本** 來產生一個新的 Docusaurus 網站。

當你執行以下指令後，classic 範本會自動加入到你的專案中：

```bash
npm init docusaurus@latest my-website classic
```
